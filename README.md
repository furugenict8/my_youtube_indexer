# my_youtube_indexer
youtubeにインデックスをつけるアプリ  
好きなyoutube動画を動画リストに追加。  
追加した動画のすきな時間にindexをつけて、アプリ上で保存できる。  

# 環境
Flutter 2.10.0  
Dart 2.16.0  

# できること
youtube動画をリストに追加  
(動画のサムネ、title表示、削除は未実装)  
リストに追加した動画にindexを追加、編集、削除
